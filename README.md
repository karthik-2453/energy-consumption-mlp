# ⚡ Energy Consumption Prediction using MLP

## 📌 Overview
This project predicts energy consumption using a Multi-Layer Perceptron (MLP) model.

## 🚀 Features
- Machine Learning prediction
- Streamlit UI
- Line graph visualization
- Smart recommendations
  
  ## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit

## ▶️ Run Project

pip install -r requirements.txt
cd frontend
python -m streamlit run app.py
